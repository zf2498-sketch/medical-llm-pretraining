# Medical Language Model Pretraining (Reproduction)

This project reproduces a domain-specific language model pretraining pipeline inspired by MedRoBERTa.

## Overview
- Fine-tuned RoBERTa on PubMed dataset
- Implemented masked language modeling (MLM)
- Compared baseline vs domain-adapted models

## Tools
- PyTorch
- Hugging Face Transformers
- Datasets

## Current Work
- Improving training efficiency
- Scaling dataset size
- Experimenting with different hyperparameters

## Goal
To explore efficient training strategies for domain-specific language models under limited computational resources.
